### [1] [Bayesian Learning-Aided Near-Field Channel Estimation for mmWave Hybrid MIMO systems employing Uniform Circular Array](https://arxiv.org/abs/2605.06039)
*Abhisha Garg,Priya Gupta,Suraj Srivastava,Aditya Jagannatham*

Main category: eess.SP

TL;DR: This paper proposes a Ring-Bayes channel learning framework that combines Bayesian learning with near-field channel estimation for mmWave hybrid MIMO systems using uniform circular arrays (UCAs), achieving superior channel recovery through a concentric-ring codebook that captures angular and distance domain features.


<details>
  <summary>Details</summary>
Motivation: As antenna numbers scale up, users fall within near-field regions where the conventional planar-wave assumption breaks down. Additionally, uniform linear arrays (ULAs) are impractical for large-scale deployment, while uniform circular arrays (UCAs) offer better beamforming gain and spatial directivity.

Method: The authors design a near-field concentric-ring codebook that captures channel features jointly in angular and distance domains, then integrate it with a Bayesian learning framework (Ring-Bayes) for UCA near-field channel estimation in hybrid MIMO systems.

Result: Extensive simulations demonstrate substantial improvements over existing methods, establishing Ring-Bayes as a powerful and scalable solution for next-generation mmWave communications.

Conclusion: Ring-Bayes is an effective, scalable framework for near-field channel estimation in large-scale mmWave MIMO systems using UCAs.

Abstract: This work conceives a Ring-Bayes channel learning framework that unifies Bayesian learning with near-field channel estimation in millimeter-wave (mmWave) hybrid MIMO systems. As the number of antennas scales up, users increasingly fall within the near-field region, rendering the conventional planar-wave assumption invalid. Moreover, the widely studied uniform linear arrays (ULAs) at the base station are impractical for large-scale deployment, whereas uniform circular arrays (UCAs) achieve superior beamforming gain and spatial directivity with the same antenna aperture. To exploit these advantages, we design a near-field concentric-ring codebook that captures channel features jointly in angular and distance domains. Leveraging this structure, the proposed Ring-Bayes framework enables highly accurate recovery of UCA near-field channels. Extensive simulations confirm that our approach delivers substantial improvements over existing methods, establishing Ring-Bayes as a powerful and scalable solution for next-generation mmWave communications.

### [2] [Robust $\mathcal{H}_\infty$ Controller Design For INDI-Controlled Quadrotor Using Online Parameter Identification](https://arxiv.org/abs/2605.05483)
*Tom Aantjes,Till M. Blaha,Spilios Theodoulis,Ewoud J. J. Smeur*

Main category: cs.RO

TL;DR: This paper designs a robust gain-scheduled attitude controller for quadrotors using INDI-based inner loop with online parameter identification, validated through simulations and flight tests showing acceptable performance even with parameters outside the defined uncertainty range.


<details>
  <summary>Details</summary>
Motivation: While INDI controller parameters can be estimated onboard within half a second during a throw, there lacks a robust method to tune outer loop gains for this feedback-linearizing controller depending on estimated model parameters.

Method: A gain-scheduled cascaded attitude controller with feedforward filter is synthesized for symmetric quadrotors using signal-based H-infinity closed-loop shaping, combined with an INDI-based inner loop and online system parameter identification.

Result: The controller exhibits good stability margins and effective tracking performance under uncertainty in nonlinear simulations. Flight tests with full online identification showed acceptable performance for actuator time constants below 40ms, even when identified parameters were far outside the defined uncertainty range.

Conclusion: The proposed robust gain-scheduled controller works effectively with online INDI parameter identification, maintaining acceptable flight performance even under significant parameter uncertainty, with a practical limit of 40ms actuator time constant.

Abstract: It has recently been shown that all physical parameters of an Incremental Nonlinear Dynamic Inversion (INDI) controller can be estimated onboard a multirotor within half a second, which is fast enough to do the full identification during a throw in the air. However, a robust method to tune outer loop gains for this feedback-linearizing INDI controller depending on the model parameters is still missing. This work presents the design of a robust gain-scheduled controller for attitude control of quadrotor, using an INDI-based inner loop with online identification of its system parameters. A gain-scheduled cascaded attitude controller with a feedforward filter is synthesized for a symmetric quadrotor using signal-based $\mathcal{H}_\infty$ closed-loop shaping. The resulting controller exhibits good stability margins, with nonlinear simulations confirming effective tracking performance under uncertainty. Experimental evaluation is also conducted through flight tests with full online parameter identification. Even though the identified parameters during these tests are far outside the defined uncertainty range, acceptable flight performance comparable to simulation results is maintained for actuator time constants below 40 ms.

</details>
